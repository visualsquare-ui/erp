# Visual Square Site

Public marketing site for Visual Square.

## Routes

- `/` public home page
- `/industries` industry landing pages
- `/blog` design insights blog
- `/robots.txt` and `/sitemap.xml`

## Deployment

Deploy this project to Vercel and connect `visualsquare.com`.

Set `NEXT_PUBLIC_SITE_URL=https://visualsquare.com` in Vercel for production.
