# Visual Square ERP Phases 2-8 MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a working internal ERP MVP through Phase 8 using Supabase data: clients, projects/tasks, work orders/proofs, assets, invoices, vendors/PO/bills, margin dashboard, and portfolio gallery.

**Architecture:** Keep all data access server-side with Supabase SSR clients and Server Actions. Implement focused list/detail pages with simple forms, revalidate affected routes after writes, and keep shared calculations in pure tested utilities.

**Tech Stack:** Next.js App Router, React Server Components, Server Actions, Supabase PostgreSQL/Auth, TypeScript, Tailwind CSS, Vitest.
